# Figma UI Components

A Vue 3 component library following Figma UI design principles.

## Features

- 📦 **Vue 3** with Composition API
- 🚀 **Vite** for fast development
- 🎨 **Figma-inspired** design system
- 📱 Responsive components
- 📝 Well-documented
- ✨ Modern CSS with scoped styles

## Project Structure

```
figma-ui-components/
├── src/
│   ├── components/          # Reusable Vue components
│   ├── composables/         # Vue composable functions
│   ├── utils/              # Utility functions
│   ├── App.vue            # Root component
│   ├── main.js            # Application entry point
│   └── style.css          # Global styles
├── index.html             # HTML template
├── vite.config.js         # Vite configuration
├── package.json           # Project dependencies
└── README.md             # This file
```

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Creating Components

1. Create a new directory in `src/components/`
2. Create a Vue component file with `.vue` extension
3. Follow the Figma design guidelines
4. Add documentation in `src/components/README.md`
