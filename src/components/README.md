# Figma UI Components

This directory contains reusable Vue 3 components designed following Figma UI principles.

## Component Structure

Each component should follow this structure:

```
ComponentName/
├── ComponentName.vue        # Main component file
├── ComponentName.stories.js # Storybook stories (if using Storybook)
├── ComponentName.test.js    # Unit tests
└── README.md                # Component documentation
```

None yet - start building your components here!

## Guidelines

1. Use Vue 3 Composition API
2. Follow Figma's design system principles
3. Maintain accessibility standards
4. Write unit tests for each component
5. Document usage and props clearly
