# Utilities

This directory contains utility functions and helpers that can be shared across the application.

## Utility Structure

Each utility should be focused on a specific task and follow this structure:

```javascript
// utilityName.js
export function utilityName(param1, param2) {
  // Logic here
}

export function anotherUtility() {
  // Logic here
}
```

## Guidelines

1. Keep utilities focused on a single concern
2. Write pure functions where possible
3. Handle edge cases
4. Add TypeScript types if using TypeScript
5. Write tests for complex utilities
6. Group related utilities into the same file
7. Use descriptive names for functions and files
